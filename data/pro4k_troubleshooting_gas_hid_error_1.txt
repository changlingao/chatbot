Category is PRO4K. Troubleshooting gas hid error step 1.
Connect printer using an ethernet cable. Continue running printer until projector HID errors occur again, then go to printer's web interface, click System Information tab and click Debug Information. Please add "?k=1" to the end of the address in the address bar (e.g. http://192.168.1.123/debug?k=1, but with your actual printer IP address instead of 192.168.1.123) and press enter. Save the complete log to file and attach to a ticket.
