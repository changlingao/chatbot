Category is Misprints & Print Defects. Prints detaching. General printer check. 
For us to check the printer, please open a support ticket and send us the printer.ini file and the build log of your printer. To get these files, please open Composer>New Build>your target printer>Blue globe icon in the top right corner, a web browser will open, please go to the tab System Information. Right-click each file and click Save Target As/Save Link As to download the text files.
