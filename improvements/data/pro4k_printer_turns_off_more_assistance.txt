If you need more help, you can open a ticket and attach the debug log/error log files to it. To get those files, open Composer, click New Build, click your target printer, then click the blue globe icon in the top right corner, a web browser will open, go to the tab System Information and you will see View debug log, and error log. Right click that file and click Save Target As/ Save Link As to download the text file.
